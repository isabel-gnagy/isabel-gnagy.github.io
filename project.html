  • Personal Portfolio Website

    Mid June 2022

    Links: Home Page | GitHub Repo

    After completing my "Personal & Professional Website" project, I realized that it had major flaws. I came to understand that I created a website tailored to my computer. Its styles and format did not keep on multi-media such as tablets or mobile views. I did not want to alter the website so I could use it as a showcase for my growth. Instead, I started from scratch and built this new site with many improvements. I believe that this is a website that I can use for an online portfolio. I hope that I have set up this website with the ability to adapt and add to it.
